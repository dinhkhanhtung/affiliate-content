2. **Nhân bản thư mục**: Sao chép thư mục dự án tương ứng từ kho tổng này ra một thư mục làm việc mới trên máy tính của bạn.
3. **Liên kết Git**: Khởi tạo Git trong thư mục mới đó và liên kết với repository GitHub riêng vừa tạo.
